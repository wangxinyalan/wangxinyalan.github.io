💻 第五周：初步认识嵌入式计算

📥 步骤1：下载Arduino IDE及安装ESP32支持包

1. 下载Arduino IDE

访问Arduino官网（https://www.arduino.cc/en/software），下载适合你操作系统的安装包。

  • Windows用户：下载.exe安装程序
  • Mac用户：下载.dmg文件
  • Linux用户：根据发行版选择对应版本

2. 安装Arduino IDE

双击安装包，按照提示完成安装。建议选择默认安装路径。

3. 安装ESP32开发板支持包

打开Arduino IDE，按以下步骤操作：

  1. 点击菜单栏 「文件」→「首选项」
  2. 在「附加开发板管理器网址」中添加：
    https://espressif.github.io/arduino-esp32/package_esp32_index.json
  3. 点击「确定」保存
  4. 点击菜单栏 「工具」→「开发板」→「开发板管理器」
  5. 在搜索框中输入 ESP32
  6. 找到 ESP32 by Espressif Systems，点击「安装」
  7. 等待安装完成（可能需要几分钟）

🎮 步骤2：熟悉Arduino IDE基本操作

界面介绍

  • 工具栏：包含验证（✓）、上传（→）、新建、打开、保存等按钮
  • 代码编辑区：编写和修改程序代码的区域
  • 控制台/输出区：显示编译和上传过程中的信息，以及串口监视器输出

常用操作

  • 验证/编译（✓）：检查代码是否有语法错误
  • 上传（→）：将代码编译并上传到开发板
  • 新建：创建一个新的空白项目（.ino文件）
  • 打开：打开已有的项目文件
  • 保存：保存当前项目
  • 串口监视器（右上角放大镜图标）：查看开发板发送的数据，调试程序

第一个测试程序：Blink（让板载LED闪烁）

// 初始化设置，只运行一次 void setup() { pinMode(LED_BUILTIN, OUTPUT); // 设置板载LED引脚为输出模式 } // 主循环，重复运行 void loop() { digitalWrite(LED_BUILTIN, HIGH); // 点亮LED delay(1000); // 等待1秒 digitalWrite(LED_BUILTIN, LOW); // 熄灭LED delay(1000); // 等待1秒 }

开发板与端口选择

  • 开发板选择：「工具」→「开发板」→「ESP32 Arduino」→选择你的ESP32型号（如ESP32 Dev Module）
  • 端口选择：「工具」→「端口」→选择正确的COM口（Windows）或/dev/cu.*（Mac）

🔌 步骤3：连接电路板与电脑，完成简单操作

硬件连接

  1. 使用USB数据线（通常是USB-A转USB-C或Micro-USB）连接ESP32开发板与电脑
  2. 观察开发板上的电源指示灯是否亮起
  3. 在Arduino IDE中检查端口是否识别到开发板

示例程序：点亮外部LED

int ledPin = 2; void setup() { pinMode(ledPin, OUTPUT); } void loop() { digitalWrite(ledPin, HIGH); delay(500); digitalWrite(ledPin, LOW); delay(500); }

示例程序：串口通信（发送"Hello World"）

void setup() { Serial.begin(115200); } void loop() { Serial.println("Hello World! 我是ESP32"); delay(1000); }
💡 常见问题与解决
- 找不到端口：检查USB线是否支持数据传输
- 上传失败：按住ESP32板上的BOOT按钮再点击上传
- 编译报错：检查开发板型号是否正确选择
