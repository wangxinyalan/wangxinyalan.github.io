📡 第六周：利用嵌入式开发完成WiFi控制舵机

🛠️ 用到的元器件

  • ESP32-S2 Mini 开发板
  • SG90 舵机（或任意 3.3V/5V 舵机）
  • 杜邦线（母对母，用于连接舵机和开发板）
  • USB 数据线（供电和上传程序）

📐 硬件接线说明

  • 舵机 VCC（红线） → ESP32-S2 Mini 5V / VIN 引脚
  • 舵机 GND（棕线） → ESP32-S2 Mini GND 引脚
  • 舵机 信号线（橙线） → ESP32-S2 Mini GPIO 2

💻 步骤1：编写 WiFi 控制舵机程序

#include <WiFi.h> #include <ESP32Servo.h> const char* ssid = "你的WiFi名称"; const char* password = "你的WiFi密码"; Servo myServo; int servoPin = 2; WiFiServer server(80); void setup() { Serial.begin(115200); myServo.attach(servoPin); myServo.write(90); WiFi.begin(ssid, password); while (WiFi.status() != WL_CONNECTED) { delay(500); Serial.print("."); } Serial.println("\nWiFi 已连接"); Serial.print("IP 地址："); Serial.println(WiFi.localIP()); server.begin(); } void loop() { WiFiClient client = server.available(); if (client) { String request = client.readStringUntil('\r'); client.flush(); if (request.indexOf("GET /set?angle=") != -1) { int start = request.indexOf("angle=") + 6; int end = request.indexOf(" ", start); String angleStr = request.substring(start, end); int angle = angleStr.toInt(); angle = constrain(angle, 0, 180); myServo.write(angle); Serial.print("设置角度："); Serial.println(angle); } client.println("HTTP/1.1 200 OK"); client.println("Content-Type: text/html"); client.println("Connection: close"); client.println(); client.println("<!DOCTYPE html><html><head><meta charset='UTF-8'><title>舵机控制</title></head><body>"); client.println("<h2>WiFi 舵机控制面板</h2>"); client.println("<input type='range' min='0' max='180' value='90' id='angleSlider' oninput='updateAngle(this.value)'>"); client.println("<p>当前角度：<span id='angleValue'>90</span>°</p>"); client.println("<script>function updateAngle(angle) { document.getElementById('angleValue').innerText = angle; fetch('/set?angle=' + angle); }</script></body></html>"); } delay(10); }

📡 步骤2：上传程序并获取 IP 地址

  1. 用 USB 数据线连接 ESP32-S2 Mini 和电脑
  2. 在 Arduino IDE 中选择对应的开发板和端口
  3. 点击「上传」按钮，将程序烧录到开发板
  4. 打开「串口监视器」（波特率 115200），等待开发板连接 WiFi 后，会显示分配到的 IP 地址
