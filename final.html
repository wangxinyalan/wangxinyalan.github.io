自动浇水系统

小组名称：天之饺子 · 本地自动 + 远程手动双控

📋 Project Profile
项目名称
自动浇水系统（本地自动 + 远程手动双控）
一句话定义
基于ESP32的智能自动浇水系统，通过土壤湿度传感器实时监测植物缺水状态，自动控制水泵浇水，并支持本地按键+远程手动/自动模式切换。
用户是谁
学校/办公室植物养护者、家庭盆栽用户、经常出差或忘记浇水的植物爱好者
使用场景
用户不在植物旁边时，系统根据土壤湿度自动浇水；用户也可通过手机/网页远程查看湿度、手动浇水或调整阈值。
HMW 挑战
如何根据土壤湿度自动且恰当地给植物浇水，同时兼顾本地与远程的灵活控制？
⚙️ 核心方案

输入： 土壤湿度传感器（模拟量）、本地按键（模式切换/手动触发）、远程指令（MQTT/HTTP）

处理逻辑： ESP32读取湿度值 → 与预设阈值比较（防抖、防干烧）→ 自动模式下继电器控制水泵启停；远程指令可覆盖自动模式

输出反馈： OLED实时显示湿度、Wi-Fi状态、模式、水泵状态；水泵抽水浇水

MVP： 读取土壤湿度 → 自动判断浇水 → 水泵工作3秒停止 → OLED显示 → 本地按键切换模式 → 远程查看控制

🔌 硬件清单
ESP32开发板 土壤湿度传感器 OLED显示屏(SSD1306) 小型水泵 3.3V继电器 按键×3 电池盒(5号电池×4) 杜邦线

结构材料： 3D打印水壶、传感器/水泵/电路板固定支架、硅胶软管、密封胶圈

// 湿度读取与滤波 int readSoilMoisture() { int raw = analogRead(SOIL_PIN); int percent = map(raw, 2900, 1500, 0, 100); percent = constrain(percent, 0, 100); // 移动平均滤波（窗口5） static int readings[5] = {0}; static int index = 0; readings[index] = percent; index = (index + 1) % 5; int sum = 0; for (int i = 0; i < 5; i++) sum += readings[i]; return sum / 5; } // 自动浇水逻辑（防抖+防干烧） void autoWatering() { static unsigned long lastWaterTime = 0; static int waterCountToday = 0; int moisture = readSoilMoisture(); if (currentMode == MODE_AUTO && moisture < THRESHOLD_DRY) { if (millis() - lastWaterTime > 300000UL && waterCountToday < 8) { digitalWrite(RELAY_PIN, HIGH); delay(3000); digitalWrite(RELAY_PIN, LOW); lastWaterTime = millis(); waterCountToday++; } } } // 远程指令处理 void callback(char* topic, byte* payload, unsigned int length) { String msg = String((char*)payload).substring(0, length); if (msg == "manual_water") { digitalWrite(RELAY_PIN, HIGH); delay(3000); digitalWrite(RELAY_PIN, LOW); } else if (msg == "auto_mode") currentMode = MODE_AUTO; else if (msg == "manual_mode") currentMode = MODE_MANUAL; }

🤔 Reflections · 硬核Bug与解决方案

✅ 软件消抖（50ms延时确认）+ 硬件电容滤波
🐛 Bug 3：MQTT连接频繁断开
✅ keepalive=30 + 自动重连机制 + WiFi断线恢复
🐛 Bug 4：OLED显示卡顿/闪烁
✅ 改为局部刷新 + 降低I2C速率至100kHz

✅ 增加连续浇水次数限制 + 空转保护（浇水后湿度无变化则禁用30分钟）
